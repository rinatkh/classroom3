# План классной работы

1. Открыть соответствующий `cmd/.../main.go`.
2. Убедиться, что `main` только вызывает `Example()`.
3. Открыть `internal/...` и читать комментарии сверху вниз.
4. Перед запуском спросить ученика: «Что выведет программа?».
5. Запустить `make run-...`.
6. Открыть unit-тест и integration-тест.

Важно: в блоках до функций не выносим логику в маленькие функции. Всё делаем внутри `Example()`, чтобы не смешивать тему функций с более ранними блоками.
