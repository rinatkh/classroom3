# Live coding script: занятие 3

## 1. Циклы

- Показать обычный `for i := 0; i < n; i++`.
- Показать while-like цикл: `for condition {}`.
- Показать `range` по слайсу и строке.
- Объяснить, почему в строках лучше идти по рунам.
- Разобрать `break` и `continue`.

## 2. Функции

- Сигнатура функции: имя, параметры, типы, возвращаемые значения.
- Несколько возвращаемых значений: `(value, ok)`.
- Variadic parameters: `numbers ...int`.
- Функция как значение: `func(int, int) int`.
- Замыкание через `MakeCounter`.

## 3. Массивы

- Длина — часть типа: `[3]int` и `[5]int` разные типы.
- Массивы копируются при присваивании и передаче в функцию.
- Массивы можно сравнивать через `==`, если элементы сравнимы.

## 4. Слайсы

- Слайс — descriptor над массивом: pointer, len, cap.
- `append` может переиспользовать старый массив или выделить новый.
- Для независимой копии использовать `copy`.
- Удаление и вставка через `append`.
